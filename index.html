Finished Projects:

Simple Servo Controller - Github

A basic Arduino sketch for controlling a motor with a potentiometer for positional feedback. The control code avoids use of a PID loop and is supposed to be easy to understand for newbies.

Drone Copy Delivery System

Ever printed off a school assignment only to realize the printer is 10 feet away and you have to get off your chair, walk across the room, grab the paper, then bring it back to your desk? Don't you wish the paper would just magically appear in front of you? Well wish no more! This Automatic Document Delivery System(TM) has you covered.

KEW Converter - Github

An application created for Cloud County Community College. They have Flash-based lessons they use for many of their wind-energy classes. Due to the impending death of Flash, I decided to look for a way to port the lessons to a more modern format. Originally the tool was intended to output a complete video/HTML5 lesson. However, much of the imagery in the lessons was made in Flash as dynamic scripts as opposed to static images, which couldn't be ripped out. Because of this, the tool only extracts what it can, and leaves it at that (so the narration and images can be re-used in an updated lesson format).

Speaker Box Robot

A basic driving robot built out of a cheapo house subwoofer. Its original iteration had a 5DOF robotic arm affixed ot the front, but the piss-poor stepper motors combined with my absolutey wonderful design skills at the time meant it couldn't really do anything useful. The wheels were also poorly built and prone to failure.



Here's some closeups of the arm and drive wheels for V1:





Later on I rebuilt the robot, removing the arm and replacing the existing wheels with more robust ones. This version, powered by an Arduino mega and Android tablet, was demoed at a local 4-H project demo, where the kids got to drive it around. It still works today. It also had a motion control function, but the code doesn't seem to work anymore.



The robot served as my introduction to ROS. I used it to control the robot over the network as well as some other fun stuff.

ServoCtrl - Github

A project to control RC Servos with 3D modelling software Blender, with use of an Arduino board. This allows for easier creation of smooth, life-like movement for animatronics and FX, since one can rig up a model of the device in Blender and animate it like any other 3D model.
